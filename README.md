# RC-CAR-ARDUINO-UNO
Arduino-based RC Car controlled via smartphone using Bluetooth communication.
## Components Used
- Arduino UNO
- HC-05 Bluetooth Module
- L298N Motor Driver
- DC Motors
- Battery Pack
- Chassis

## Features
- Forward Movement
- Backward Movement
- Left Turn
- Right Turn
- Stop

## Control Method
The RC car is controlled wirelessly using a smartphone Bluetooth application.
